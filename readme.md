# Cuda Rust Templates 
This repo contains a portable Rust pipeline for Cuda kernel development and task delegation.

# Projects
1. Simple Addition and Multiplication Kernel to test the Pipeline
2. LONG Liquidation Kernel with fixed positions list and ticker price (currently just one pair per kernel, soon more)


# Development and Growth
Every line of code in this repo was written by hand and AI was only used for assistance in learning CUDA.
The motivation behind these efforts is to be able to contribute more to high-throughput, scalable solutions
that require parallel computation and clever scheduling.
